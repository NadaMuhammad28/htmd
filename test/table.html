Trouble code OBD/EOBD
DTC 100 - 199
AUDI; A3 (8P1); 1.2 TSI; kW: 77
EOBD - diagnostic connection

Data format per ISO 9141-2, SAE J 1850 or SAE J 2284
The diagnostic connection is located inside the vehicle and must be accessible from the driver's seat.

OBD/EOBD component position

Connection configuration
PIN 1 Assigned, manufacturer-specific
PIN 2 Data line Plus (SAE J 1850)
PIN 3 Assigned, manufacturer-specific
PIN 4 Vehicle ground
PIN 5 Signal dimensions
PIN 6 Data line CAN High (SAE J 2284)
PIN 7 K-line (ISO 9141-2)
PIN 8 Assigned, manufacturer-specific
PIN 9 Assigned, manufacturer-specific
PIN 10 Data line Minus (SAE J 1850)
PIN 11 Assigned, manufacturer-specific
PIN 12 Assigned, manufacturer-specific
PIN 13 Assigned, manufacturer-specific
PIN 14 Data line CAN Low (SAE J 2284)
PIN 15 L-line (ISO 9141-2)
PIN 16 Battery plus


Freeze Frame Data

(Frame data which is saved when failure occurs)

- Engine load

- Coolant temperature 		- Fuel regulation system status, regulated/unregulated
- Ignition time adjustment
- Intake air temperature
- Driving speed
- Intake-manifold pressure
- Air flow
- Throttle valve setting
- Secondary air status
- Trouble code OBD/EOBD
- Engine oil temperature
